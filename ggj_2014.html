Global Game Jam 2014

Game Overview

  • Game made for the 2014 Global Game Jam
  • First player to 10 points wins; obtain 1 point for each powerup collected (treasure chests for yellow player, skulls for red player)
  • Each player can throw a weapon; if it hits the opponent they lose 1 point, if it hits a powerup then the powerup gets destroyed
  • The opponent's powerups don't show up on your screen but the light they project on the ground does
  • Can play with two controllers or even one keyboard (WASD and spacebar for player 1, arrow keys and right ctrl for player 2)

Technical Overview

  • Languages: C#
  • Tools: Unity, Blender (modelling, rigging, texturing and animating), ProBuilder (level creation)
  • Platforms: Windows, Mac, Linux

Notes

Because this project uses paid assets (Toony Colors Pro, ProBuilder) I only provide the builds for download and not the project itself
