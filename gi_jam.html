G.I. Jam

Game Overview

  • Game made for a game jam held by the University of Waterloo Games Institute
  • Collect all the gold items in a level by flicking a shuriken and having it bounce off walls
  • Every collision with a wall costs 1 energy, collecting a green item restores 1 energy
  • For each level you start with a set amount of energy, if you reach 0 energy before collecting all the gold items it's game over

Technical Overview

  • Languages: C#
  • Libraries: Touchscript, NGUI
  • Tools: Unity, Blender (modelling), ProBuilder (level creation)
  • Platforms: Android, Windows

Notes

Because this project uses paid assets (NGUI, ProBuilder) I only provide the builds for download and not the project itself
