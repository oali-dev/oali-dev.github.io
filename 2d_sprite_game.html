2D Sprite Game Prototype

Game Overview

  • Press 1, 2 or 3 to switch between character sprites
  • Press 'A' to load sewer test map
  • Press 'S' to load cave test map
  • Press Esc to exit

Technical Overview

  • Languages: C++
  • Libraries: SFML, TinyXML
  • Notable Features: Level file parsing and loading, Tile-based character movement with collisions
  • Tools: Tiled (level creation using tile sets)
  • Platforms: Windows
